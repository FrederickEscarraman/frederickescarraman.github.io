Etiqueta Nombre Descripcion
<!-- --> Etiqueta de Comentario La etiqueta de comentario se utiliza para insertar comentarios en el código fuente. Los comentarios no se muestran en los navegadores.
<!DOCTYPE html> Etiqueta DOCTYPE Todos los documentos HTML deben comenzar con una declaración <!DOCTYPE>. La declaración no es una etiqueta HTML, si no es solo información para que el navegador pueda saber qué tipo de documento esperar.
<a> Etiqueta de Hipervinculo La etiqueta define un hipervínculo, que se utiliza para redireccionar al usuario de una página a otra. El atributo más importante de la etiqueta es el atributo href, que indica el destino del enlace.
<abbr> Etiqueta de Abreviatura La etiqueta define una abreviatura o un acrónimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP" o "ATM".
<address> Etiqueta de Direccion La etiqueta define la información de contacto del autor de un documento o artículo. Esta puede ser una dirección de correo electrónico, URL, dirección física, número de teléfono, ID de redes sociales, etc.
<area> Etiqueta de Area La etiqueta define un área dentro de un mapa de imagen (un mapa de imagen es una imagen con áreas en las que se puede hacer clic). Las imagenes siempre deben estar dentro de la etiqueta area.
<article> Etiqueta de Articulo La etiqueta especifica contenido independiente y que se contiene a si mismo. Un artículo debe tener sentido por sí mismo y debe ser posible distribuirlo independientemente del resto del sitio.
<aside> Etiqueta al Lado La etiqueta define algún contenido además del contenido en si que se coloca. El contenido al lado debe estar indirectamente relacionado con el contenido principal.
<audio> Etiqueta de Audio Se utiliza para incrustar contenido de sonido en un documento, como música o transmisiones de audio. La etiqueta contiene una o más etiquetas con diferentes fuentes de audio. El navegador elegirá la primera fuente que admita.
<b> Etiqueta de texto en Negrita La etiqueta especifica texto en negrita sin ninguna importancia adicional.
<base> Etiqueta de URL Base Especifica la URL base o el destino de todas las URL relativas en un documento. La etiqueta debe tener presente un atributo href.
<bdi> Etiqueta de Aislamiento bidireccional La etiqueta aísla una parte del texto que podría tener un formato en una dirección diferente de otro texto fuera de ella. Este elemento es útil cuando se incrusta contenido generado por el usuario con una dirección de texto desconocida.
<bdo> Etiqueta de anulacion bidireccional La etiqueta se utiliza para anular la dirección del texto actual.
<blockquote> Etiqueta de seccion de citas La etiqueta especifica una sección que se cita de otra fuente. Es representado usualmente como una sangria.
<body> Etiqueta de cuerpo del documento El elemento contiene todo el contenido de un documento HTML, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.
<br> Etiqueta de salto de linea La etiqueta es útil para separar texto o elementos unos de otros. Es una etiqueta vacía, lo que significa que no tiene una etiqueta final.
<button> Etiqueta de boton La etiqueta define un botón en el que se puede hacer clic y realizar acciones por medio de este evento. Dentro de un elemento puede ser asignado texto.
<canvas> Etiqueta de lienzo La etiqueta se usa para dibujar gráficos, durante la ejecucion del sitio web, a través de codigo fuente (generalmente JavaScript). La etiqueta es transparente y es solo un contenedor para gráficos, debe usar un script para dibujar los gráficos.
<caption> Etiqueta de titulo de Tabla La etiqueta define un título de tabla. La etiqueta debe insertarse inmediatamente después de la etiqueta table.
<cite> Etiqueta de titulo de Tabla La etiqueta define el título de una obra creativa (por ejemplo, un libro, un poema, una canción, una película, una pintura, una escultura, etc.).
<code> Etiqueta de codigo La etiqueta se utiliza para definir una pieza de código informatico. El contenido del interior se muestra en la fuente monoespaciada predeterminada del navegador.
<col> Etiqueta de columna La etiqueta especifica las propiedades de columna para cada columna dentro de un elemento colgroup. La etiqueta es útil para aplicar estilos a columnas enteras, en lugar de repetir los estilos para cada celda y para cada fila.
<colgroup> Etiqueta de grupo de columnas La etiqueta especifica un grupo de una o más columnas en una tabla. La etiqueta es útil para aplicar estilos a columnas enteras, en lugar de repetir los estilos para cada celda y para cada fila.
<data> Etiqueta de datos La etiqueta se utiliza para agregar una traducción que sea legible por la máquina de un contenido determinado. Este elemento proporciona un valor legible para los procesadores de datos y un valor legible por humanos para la representación en un navegador.
<datalist> Etiqueta lista de datos La etiqueta especifica una lista de opciones predefinidas para un elemento. La etiqueta se utiliza para proporcionar una función de autocompletar para los elementos.
<dd> Etiqueta dd La etiqueta se utiliza para describir un término/nombre en una lista de descripción. Se usa junto con dl (define una lista de descripciones) y dt (define términos/nombres). Dentro de una etiqueta se pueden usar párrafos, saltos de línea, imágenes, enlaces, listas, etc.
<del> Etiqueta eliminado La etiqueta define un texto que se ha eliminado de un documento. Los navegadores generalmente marcarán una línea a través del texto eliminado.
<details> Etiqueta de detalles La etiqueta especifica detalles adicionales que el usuario puede abrir y cerrar a gusto. La etiqueta se usa a menudo para crear un widget interactivo que el usuario puede abrir y cerrar. De forma predeterminada, el widget está cerrado. Cuando está abierto, se expande y muestra el contenido que contiene. Cualquier tipo de contenido se puede poner dentro de la etiqueta.
<dialog> Etiqueta de dialogo La etiqueta define un cuadro de diálogo o una subventana. El elemento facilita la creación de cuadros de diálogo emergentes y modales en una página web.
<div> Etiqueta div La etiqueta define una división o una sección en un documento HTML. La etiqueta se utiliza como contenedor de elementos HTML, que luego se diseñan con CSS o se manipulan con JavaScript.
<dl> Etiqueta de lista de descripcion La etiqueta define una lista de descripción. La etiqueta se usa junto con dt (define términos/nombres) y dd (describe cada término/nombre).
<dt> Etiqueta de termino de lista de descripcion La etiqueta define un término/nombre en una lista de descripción. La etiqueta se usa junto con dl (define una lista de descripción) y dd (describe cada término/nombre).
<em> Etiqueta em La etiqueta se utiliza para definir texto enfatizado. El contenido interior normalmente se muestra en cursiva. Un lector de texto en pantalla puede pronunciar las palabras con énfasis, usando acento verbal.
<embed> Etiqueta codigo incrustado La etiqueta define un contenedor para un recurso externo, como una página web, una imagen, un reproductor multimedia o una aplicación complementaria.
<fieldset> Etiqueta de conjunto de campos La etiqueta se utiliza para agrupar elementos relacionados en un formulario. La etiqueta dibuja un cuadro alrededor de los elementos relacionados.
<figcaption> Etiqueta figcaption La etiqueta define un título para un elemento figure. El elemento se puede colocar como el primer o último hijo del elemento figure.
<figure> Etiqueta de figura La etiqueta especifica contenido independiente, como ilustraciones, diagramas, fotos, listas de códigos, etc. Si bien el contenido del elemento está relacionado con el flujo principal, su posición es independiente del flujo principal.
<footer> Etiqueta de pie de pagina La etiqueta define un pie de página para un documento o sección. Un elemento normalmente contiene: información de autoría, informacion registrada, Información del contacto, mapa del sitio, volver a los enlaces superiores y documentos relacionados.
<form> Etiqueta de formulario La etiqueta se utiliza para crear un formulario HTML para ingresar datos del usuario.
<h1> - <h6> Etiqueta de encabezado Las etiquetas h1 a h6 se utilizan para definir encabezados HTML. h1 define el encabezado más importante. h6 define el encabezado menos importante.
<head> Etiqueta de encabezado del documento El elemento es un contenedor de metadatos y se coloca entre la etiqueta html y la etiqueta body. Los metadatos son datos sobre el documento HTML. Los metadatos no se muestran, solo suelen definir el título del documento.
<header> Etiqueta de conjunto de enlaces El elemento representa un contenedor de contenido introductorio o un conjunto de enlaces de navegación. Un elemento normalmente contiene: uno o más elementos de encabezado (h1 - h6), un logotipo o icono y algo de información basica.
<hr> Etiqueta de ruptura tematica La etiqueta define una ruptura temática en una página HTML (por ejemplo, un cambio de tema). El elemento suele mostrarse como una regla horizontal que se usa para separar el contenido (o definir un cambio) en una página HTML.
<html> Etiqueta de documento HTML La etiqueta representa la raíz de un documento HTML. La etiqueta es el contenedor de todos los demás elementos HTML (excepto la etiqueta !DOCTYPE).
<i> Etiqueta de cursiva El contenido se muestra en cursiva. La etiqueta se usa a menudo para indicar un término técnico, una frase de otro idioma, un pensamiento, el nombre de un barco, etc.
<iframe> Etiqueta de cursiva La etiqueta especifica una area incrustada. Estas areas se utilizan para incrustar otro documento dentro del documento HTML actual.
<img> Etiqueta de Imagen La etiqueta se utiliza para implementar una imagen en una página HTML. Técnicamente, las imágenes no se insertan en una página web, estas están vinculadas a páginas web. La etiqueta crea un espacio de espera para la imagen de referencia.
<input> Etiqueta de entrada de datos La etiqueta especifica un campo de entrada donde el usuario puede ingresar datos. El elemento es el elemento más importante en un formulario. Se puede mostrar de varias formas, según el atributo de tipo.
<ins> Etiqueta de insercion La etiqueta define un texto que ha de ser subrayado.
<kbd> Etiqueta kbd La etiqueta se utiliza para definir texto que ingresa el teclado. El contenido del interior se muestra en la fuente monoespaciada predeterminada del navegador.
<label> Etiqueta de etiqueta La etiqueta define una etiqueta para elementos de formulario. El uso adecuado de las etiquetas con los formularios ayuda a los usuarios que tienen dificultades para hacer clic en regiones muy pequeñas (como casillas de verificación).
<legend> Etiqueta de leyenda La etiqueta define un título para el elemento fieldset.
